# Fortumars SaaS Platform

This is a modern SaaS starter built with Next.js 15, React, Tailwind CSS, ShadCN UI, and Firebase.

## Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Pushing to GitHub

Follow these steps to push your project to GitHub:

1. **Create a new repository on GitHub**: Go to [github.com/new](https://github.com/new) and create a repository without initializing it with any files.
2. **Open your terminal** in the project root.
3. **Initialize Git**:
   ```bash
   git init
   ```
4. **Stage your files**:
   ```bash
   git add .
   ```
5. **Commit your changes**:
   ```bash
   git commit -m "Initial commit"
   ```
6. **Branch to main**:
   ```bash
   git branch -M main
   ```
7. **Add the remote origin**:
   Replace `<YOUR_REPO_URL>` with your actual repository URL (e.g., `https://github.com/username/repo-name.git`).
   ```bash
   git remote add origin <YOUR_REPO_URL>
   ```
8. **Push to GitHub**:
   ```bash
   git push -u origin main
   ```

- **Components**: ShadCN UI
- **Database/Auth**: Firebase
