# **App Name**: Fortumars SaaS Platform

## Core Features:

- Homepage Hero Section: Dynamic hero section with bold headline, value-driven subheadline, and clear CTAs, designed for high conversion rates.
- SaaS Pricing Cards: Interactive pricing cards for Basic, Pro, and Enterprise plans, with feature comparison table and clear CTA buttons for upgrading.
- Contact Form Submission: Collect user details through the contact form and store the information securely in Firestore.
- User Authentication: Firebase Authentication integration for secure user login and signup, with email verification and password reset functionality.
- Personalized Feature Explanations: Generate tailored explanations of key features based on the identified business type of the user (identified by email)
- Trust Building: Customer trust section featuring company logos and dynamically generated testimonials, enhanced by AI to align with each user.
- Product pages: Informative product pages, including one main product page and additional 'Features' page with clear explanations, benefits over competitors, and use cases.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for trust and innovation.
- Background color: Off-white (#FAFAFA) for a clean, modern look.
- Accent color: Teal (#008080) for highlighting important elements and CTAs.
- Font: 'Inter' (sans-serif) for both headlines and body text. Note: currently only Google Fonts are supported.
- Modern and minimalistic icons to represent various features and benefits, enhancing visual appeal.
- Clean, organized layout with ample white space to improve readability and focus on key information.
- Subtle hover effects and smooth transitions for a professional and engaging user experience.
